1- The Software Development Life Cycle (SDLC) i
    - Requirement Analysis  - Gather and analyze customer requirements.
    - Planning  - Estimate cost, resources, and timeline.
    - System Design    - Create system architecture and database design.
    - Implementation (Coding)    - Developers write the source code based on the design.
    - Testing  - Verify that the software meets requirements.
    - Deployment    - Release the software to users.
    - Maintainance   - Fix bugs and security issues.
